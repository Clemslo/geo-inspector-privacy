Politique de confidentialité — GEO Inspector

Dernière mise à jour : [À COMPLÉTER — date de publication]

Cette politique de confidentialité décrit comment l'extension Chrome GEO Inspector (ci-après « l'extension ») traite les données lorsque vous l'utilisez.

1. Ce que fait l'extension

GEO Inspector analyse la page web actuellement ouverte dans votre onglet actif afin de produire un audit de bonnes pratiques SEO/GEO (optimisation pour la recherche par IA générative), basé sur le guide officiel de Google. L'analyse est déclenchée uniquement lorsque vous cliquez sur l'icône de l'extension dans la barre d'outils.

2. Données traitées

Pour produire l'audit, l'extension lit certaines informations de la page active :

  • Contenu de la page : titre, balises meta, titres (H1-H6), paragraphes, images, liens, données structurées (JSON-LD), formulaires.
  • Fichiers publics du domaine consulté : robots.txt, sitemap.xml, llms.txt, ainsi que les en-têtes HTTP de la page.

Ce traitement est effectué localement, dans votre navigateur. Aucune de ces informations n'est envoyée à un serveur appartenant au développeur de l'extension, ni à aucun tiers.

Les seules requêtes réseau sortantes effectuées par l'extension sont adressées au même domaine que celui affiché dans l'onglet actif (pour lire robots.txt, sitemap.xml, llms.txt et les en-têtes HTTP de la page) — jamais vers un serveur tiers ou vers une infrastructure appartenant au développeur.

3. Données que nous NE collectons PAS

L'extension ne collecte, ne stocke et ne transmet à aucun moment :

  • d'informations permettant de vous identifier personnellement (nom, adresse, e-mail, etc.) ;
  • d'informations de santé, financières ou de paiement ;
  • de mots de passe ou d'informations d'authentification ;
  • de communications personnelles (e-mails, SMS, messages) ;
  • de données de localisation ;
  • de votre historique de navigation ;
  • de votre activité (clics, position de la souris, frappes clavier).

4. Stockage local

L'extension utilise l'API chrome.storage.local pour mémoriser une seule information : votre préférence d'affichage (thème clair ou sombre). Cette information reste stockée localement sur votre appareil et n'est jamais transmise.

5. Presse-papiers

Si vous cliquez sur le bouton « Copier le rapport », l'extension copie le résumé de l'audit (généré localement) dans votre presse-papiers, via l'autorisation clipboardWrite. Cette action ne se déclenche que sur votre clic explicite.

6. Partage avec des tiers

Nous ne vendons, ne louons ni ne partageons aucune donnée utilisateur avec des tiers. Nous n'utilisons aucune donnée à des fins publicitaires, de profilage, de score de solvabilité ou de prêt.

7. Sécurité

Comme l'ensemble du traitement est local et qu'aucune donnée n'est transmise à une infrastructure externe, il n'existe pas de base de données serveur contenant vos informations qui pourrait être compromise.

8. Modifications de cette politique

Cette politique peut être mise à jour si les fonctionnalités de l'extension évoluent. La version en vigueur est toujours disponible à l'URL indiquée sur la fiche Chrome Web Store de l'extension.
